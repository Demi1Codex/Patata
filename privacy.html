Política de Privacidad y Protección de Datos - Patata

Última actualización: 20 de diciembre de 2025

En Borrachos.docx (https://demi1codex.github.io/Patata/), la privacidad de nuestros usuarios es una prioridad. Este documento describe cómo recopilamos, usamos y protegemos su información, especialmente en relación con la integración de los servicios de Google.

1. Información que recopilamos

Borrachos.docx es una herramienta diseñada para la gestión de eventos y organización personal. Para funcionar correctamente, la aplicación solicita acceso a su cuenta de Google a través de los servicios de OAuth2. Solo recopilamos la información estrictamente necesaria para la operatividad de la herramienta.

2. Uso de Google Calendar y Permisos

Nuestra aplicación utiliza la API de Google Calendar para permitir una sincronización bidireccional de sus actividades. Al utilizar Borrachos.docx, usted otorga el siguiente permiso:

Ver, editar, compartir y eliminar permanentemente todos los calendarios a los que puedes acceder mediante Google Calendar.

Reafirmación del uso:

  • Lectura: Borrachos.docx accede a sus eventos existentes para mostrarlos dentro de nuestra interfaz y ayudarle a evitar conflictos de horario.
  • Escritura/Edición: La aplicación permite crear nuevos eventos o modificar los existentes directamente desde nuestra plataforma para facilitar su flujo de trabajo.
  • Gestión: El permiso de "eliminar" se requiere técnicamente para permitirle borrar eventos que usted mismo decida descartar a través de nuestra interfaz.

3. Almacenamiento y Seguridad de Datos

  • Acceso limitado: Borrachos.docx no almacena una copia de su base de datos de Google Calendar en servidores externos de forma permanente. La interacción ocurre principalmente en el lado del cliente (browser) o mediante llamadas directas a la API de Google.
  • Tokens de acceso: Los tokens de acceso proporcionados por Google se manejan de forma segura y se utilizan exclusivamente para autenticar sus solicitudes durante la sesión de uso.
  • No comercialización: Bajo ninguna circunstancia vendemos, alquilamos ni compartimos sus datos de calendario con terceros ni con empresas de publicidad.

4. Limitación de Uso (Política de Datos de Usuario de Google)

El uso y la transferencia de la información recibida de las APIs de Google por parte de Patata se adherirán a la Política de Datos de Usuario de los Servicios de API de Google, incluidos los requisitos de Uso Limitado. Solo utilizamos los datos para proporcionar o mejorar las funciones que son prominentes en la interfaz de usuario de nuestra aplicación.

5. Control del Usuario

Usted tiene el control total sobre sus datos:

  • Puede revocar el acceso de Borrachos.docx a su cuenta de Google en cualquier momento a través de la Configuración de Seguridad de su Cuenta de Google.
  • Al revocar el acceso, Borrachos.docx dejará de tener capacidad para leer o modificar sus calendarios de forma inmediata.

6. Contacto

Si tiene preguntas sobre esta política o sobre cómo manejamos sus datos, puede contactarnos a través del repositorio oficial en GitHub o mediante los canales de contacto habilitados en la web.
